# Builder :v:

## Установить зависимости
```bash
$ npm install
```

## Запуск в режиме dev
```bash
$ npm run dev
```

## Сборка проекта в режиме prod
```bash
$ npm run prod
```

## Запуск собранного проекта в режиме prod
```bash
$ npm run serve
```

---

## Создание нового проекта

```bash
$ mkdir project_name
$ cd project_name
$ git clone https://github.com/
$ rm -rf .git
$ git init
$ npm install
```
